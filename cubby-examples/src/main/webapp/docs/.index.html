Cubby - Simple JSP based Web Application Framework

What's Cubby?

Cuuby(カビー)はJSPをベースにしたシンプルなWebアプリケーションフレームワークです。 以下の特徴があります。

  • 100% Pure JSP Web Application Framework
  • One Configuration
  • シンプル＆スモール
  • RESTっぽいURIをサポート(/agata/todo/1001,拡張子が付かないURIなど)
  • 直感的な名前のタグライブラリ
