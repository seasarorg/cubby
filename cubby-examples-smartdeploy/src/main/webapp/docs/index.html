Cubby

What's Cubby?

Cubby(カビー)はJava/JSPをベースにしたシンプルなWebアプリケーションフレームワークです。
以下の特徴があります。
  • 100% Pure JSP Web Application Framework
  • One Configuration
  • シンプル＆スモール
  • RESTっぽいURIをサポート(/username/todo/1001,拡張子が付かないURIなど)
  • 直感的な名前のタグライブラリ
