What's Cubby?

Cuuby(カビー)はJSPをベースにしたシンプルなWebアプリケーションフレームワークです。 以下の特徴があります。

  • 100% Pure JSP Web Application Framework
  • One Configuration
  • シンプル＆スモール
  • RESTっぽいURIをサポート(/agata/todo/1001,拡張子が付かないURIなど)
  • 直感的な名前のタグライブラリ

動作環境

  • Java5以上
  • Servlet2.4/JSP2.0以上
  • Seasar2.4.8以上

セットアップ

  1. cubby-x.x.x.jarと依存ライブラリをクラスパスに追加。
    依存ライブラリ
    • commons-beanutils1.7.0.jar
    • commons-fileupload1.1.1.jar
    • commons-io-1.1.jar
    • commons-lang-2.2.jar
    • commons-logging-1.1.jar
    • jstl-1.1.2.jar
    • s2-extension-2.4.8.jar
      • s2-framework-2.4.8.jar
      • aopalliance-1.0.jar
      • log4j-1.2.8.jar
      • javassist-3.4.ga.jar
      • ognl-2.6.7.jar
  2. web.xmlにcubbyとseasar2とServletFilterの設定を追加。
    web.xml 
      <filter>
    <filter-name>encodingfilter</filter-name>
    <filter-class>org.seasar.extension.filter.EncodingFilter</filter-class>
    <init-param>
      <param-name>encoding</param-name>
      <param-value>UTF-8</param-value>
    </init-param>
  </filter>
  <filter>
    <filter-name>s2filter</filter-name>
    <filter-class>org.seasar.framework.container.filter.S2ContainerFilter</filter-class>
  </filter>
  <filter>
    <filter-name>cubbyFilter</filter-name>
    <filter-class>org.seasar.cubby.servlet.CubbyFilter</filter-class>
  </filter>
  <filter-mapping>  
    <filter-name>encodingfilter</filter-name> 
    <url-pattern>/*</url-pattern>  
  </filter-mapping> 
  <filter-mapping>
    <filter-name>s2filter</filter-name>
    <url-pattern>/*</url-pattern>
  </filter-mapping>
  <filter-mapping>
    <filter-name>cubbyFilter</filter-name>
    <url-pattern>/*</url-pattern>
  </filter-mapping>
  <filter-mapping>
    <filter-name>cubbyFilter</filter-name>
    <url-pattern>/cubby-dispacher/*</url-pattern>
    <dispatcher>FORWARD</dispatcher>
  </filter-mapping>
  3. app.diconでcubby.diconをincludeします。
    app.dicon 
    <?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE components PUBLIC "-//SEASAR//DTD S2Container 2.3//EN"
"http://www.seasar.org/dtd/components23.dtd">
<components namespace="app">
    <include path="cubby.dicon" />
    <include path="allservice.dicon" />
</components>
