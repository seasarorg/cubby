Cubby tags
Tag select

HTML の select タグと option タグを出力します。

Example: 
(JSP) <t:select name="typeId" items="${todoTypes}" labelProperty="name" valueProperty="id"/> (実行結果) <select name="typeId"> <option name="1">TodoType 1</option> <option name="2">TodoType 2</option> </select>

nametruetruejava.lang.Stringselected 属性を付加する option タグを決めるための値を取得するプロパティ名を指定します。 この属性の値は name 属性としてそのまま出力されます。
itemstruetruejava.lang.Objectoption タグの内容を表すリストを指定します。 配列や java.util.Collection、java.util.Map 型が指定できます。
labelPropertyfalsetruejava.lang.Stringitems 属性で指定したリストの要素から、option タグの表示文字列として使用するプロパティ名を指定します。 items 属性で指定した型が java.util.Map の場合は無視されます。
valuePropertyfalsetruejava.lang.Stringitems 属性で指定したリストの要素から、option タグの値として使用するプロパティ名を指定します。 items 属性で指定した型が java.util.Map の場合は無視されます。
emptyOptionfalsetruejava.lang.Boolean空白の option タグを出力するかどうかを指定します。 省略した場合は true と見なして先頭に空白の option タグが出力されます。
emptyOptionLabelfalsetruejava.lang.String空白の option タグの表示文字列を指定します。 省略した場合は何も表示されません。
