プロジェクト情報

この文書は、このプロジェクトの一般的な情報の一部である様々な文書とリンクの概要を示します。すべてのコンテンツはこのプロジェクトのために Maven によって生成されたものです。

概要

文書説明
このプロジェクトについてCubby Simple JSP based Web Application Framework
ソースリポジトリウェブブラウザによって閲覧できるオンラインソースリポジトリへのリンクです。
プロジェクトサマリこのプロジェクトの関連情報です。
プロジェクトチームこの文書はプロジェクトのメンバの情報を提供します。これらは、何らかの形でプロジェクトに貢献した人たちです。
プロジェクトライセンスプロジェクトライセンスの定義へのリンクです。
メーリングリストこの文書はこのプロジェクトのメーリングリストの購読方法と、メーリングリストのアーカイブに関する情報を提供します。
依存関係この文書はプロジェクトの依存ライブラリをリストアップし、個々の依存ライブラリの情報を提供します。
依存関係の一致度この文書はプロジェクト全体とそのサブモジュールに渡って、一貫して同じバージョンの依存関係が使われているかどうかを示します。
課題追跡このプロジェクトの課題管理システムへリンクです。このリンクを利用して課題 (バグ、機能、変更要求) を作成し、検索できます。
継続的結合コードのビルドとテストを定期的かつ頻繁に行う、すべての継続的結合プロセスの定義へのリンクです。
